Iris Predictor Application

Project Overview

This application uses the Iris dataset to predict the flower species based on user-provided input. The prediction is powered by a machine learning model trained with WEKA using a Multi-Layer Perceptron (MLP). Users can interact through a web interface to input sepal and petal length/width values and receive predictions.

Technologies Used

  • Java Spring Boot: For backend development.
  • WEKA: For training the machine learning model.
  • Thymeleaf: For web interface development.
  • Maven: For dependency management.

Installation Instructions

Clone the Repository:

git clone https://github.com/username/iris-predictor.git
cd iris-predictor

Install Dependencies:

Use Maven to install the dependencies:

mvn clean install

Run the Application:

mvn spring-boot:run

Access the Web Interface:

Open your browser and go to http://localhost:8080.

How to Use?

  1. Access the web interface.
  2. Enter sepal and petal length/width values.
  3. Click on the Predict button.
  4. The predicted flower species will be displayed below the form.
