Iris Tahmin Uygulaması (Iris Predictor)

Proje Tanıtımı

Bu uygulama, Iris veri setini kullanarak bir çiçek türünü tahmin etmek için bir makine öğrenimi modeli kullanır. Uygulama, WEKA kullanılarak eğitilmiş bir MLP (Multi-Layer Perceptron) modeline dayanmaktadır. Kullanıcılar, web arayüzü aracılığıyla çiçeğin sepal ve petal uzunluk/genişlik değerlerini girerek tahmin sonuçlarını görüntüleyebilirler.

Kullanılan Teknolojiler

  • Java Spring Boot: Backend geliştirme için kullanılmıştır.
  • WEKA: Makine öğrenimi modeli eğitimi için kullanılmıştır.
  • Thymeleaf: Web arayüzü geliştirme için kullanılmıştır.
  • Maven: Proje bağımlılıklarını yönetmek için kullanılmıştır.

Kurulum Talimatları

Depoyu Klonlayın:

git clone https://github.com/kullaniciadi/iris-predictor.git
cd iris-predictor

Bağımlılıkları Yükleyin:

Maven kullanarak bağımlılıkları yükleyin:

mvn clean install

Uygulamayı Çalıştırın:

mvn spring-boot:run

Web Arayüzüne Erişin:

Tarayıcınızda http://localhost:8080 adresine gidin.

Nasıl Kullanılır?

  1. Web arayüzüne erişin.
  2. Sepal ve petal uzunluk/genişlik değerlerini girin.
  3. Tahmin Et butonuna tıklayın.
  4. Tahmin edilen çiçek türü, formun altında görüntülenecektir.
